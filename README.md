# AI-Powered Growth Analyst Frontend

This is the frontend for an AI-Powered Growth Analyst application built with Next.js. It provides a chat interface to interact with an AI model, manage conversation history, and configure user settings.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set up your environment variables by copying `.env.example` to `.env.local`:
   ```bash
   cp .env.example .env.local
   ```

3. Fill in your credentials in `.env.local`

4. Run the development server:
   ```bash
   npm run dev
   ```

   Open [http://localhost:3000](http://localhost:3000) to start chatting.

## What's Inside

- **Chat interface** - Talk to the AI growth analyst
- **History** - Browse past conversations
- **Settings** - Configure your preferences
- **OAuth support** - Sign in with Microsoft and connect to Google accounts
